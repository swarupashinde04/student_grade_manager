# Student Grade Manager

A simple Python mini project that helps manage student marks using a menu-driven program.

## Description

This project is built using basic Python concepts and is suitable for beginners.  
The program allows users to add student names and marks, view all students, and search for a specific student.

It runs in the terminal and demonstrates the use of:

- Dictionary
- Loops
- Conditional statements
- User input
- Menu-driven programming

## Features

- Add student and marks
- View all students
- Search student by name
- Exit program

## Language Used

Python

## How to Run

python student_grade_manager.py
